# paw skills

Seven domain knowledge bundles. Each loads on demand when the request matches.

| Skill | Triggers when |
|---|---|
| [architecture](architecture/SKILL.md) | Designing systems, evaluating tradeoffs, component boundaries |
| [devils-advocate](devils-advocate/SKILL.md) | Reviewing any plan before commitment |
| [frontend](frontend/SKILL.md) | React, Vue, or Angular work |
| [php-laravel](php-laravel/SKILL.md) | PHP or Laravel work |
| [security](security/SKILL.md) | Code touching auth, user input, external APIs, sensitive data |
| [quality-gate](quality-gate/SKILL.md) | Reviewing code before merge |
| [git-safety](git-safety/SKILL.md) | Any git operation that could lose work |

Each skill works standalone — read the SKILL.md directly for guidance outside of paw agents.
